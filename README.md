# Ethereum Activity Analysis: Participation vs Engagement

Research Question

Is Ethereum’s growth driven more by participation (unique addresses) or engagement (logic transactions)?

Metrics Deployed

1. Total Transactions → measures overall activity
2. Logic vs Value Transactions → measures engagement(contract calls,mints,swaps,etc) vs ETH transfers
3. Unique Addresses → measures participation

Methodology

Data was sourced from the Ethereum transactions table on Dune Analytics.
Transactions were grouped by day over a 14-day period.


 Key Observations

1. Value transactions were consistently higher than logic transactions
2. Unique addresses fluctuated without a clear upward trend
3. Total transaction activity varied across the period

 Interpretation

The data suggests that ETH transfer activity dominates over smart contract interactions within the observed window.

---

 Limitations

1. Short time frame (14 days) may not reflect long-term trends
2. Value transactions may include non-meaningful activity (e.g., exchange transfers)
3. Unique addresses do not perfectly represent actual users


 Conclusion

There is no clear evidence that Ethereum’s growth is driven by either participation or engagement within this dataset.
